# 6048: [기초-출력] 출력하기48(설명)(py)
## 문제설명
```
두 정수(a, b)를 입력받아
a와 b의 값이 같으면 True 를, 같지 않으면 False 를 출력하는 프로그램을 작성해보자.
```
***
## 출력 예시
~~~
True
~~~
***
## 내 풀이
```python
a, b = input().split() 
a = int(a) 
b = int(b) 
print(a==b)

````
***
## 모범 답안
