# 6008: [기초-출력] 출력하기08(설명)(py)
## 문제설명
```
정수(integer)는
양의 정수(1, 2, 3, 4, 5, ...), 음의 정수(-1, -2, -3, -4, -5, ...), 0 과 같이
소숫점 아래에 수가 없는 수라고 할 수 있다.

변수에 정수값을 저장한 후 정수로 변환하여 출력해보자.
```
***
## 출력 예시
~~~
15
~~~
***
## 내 풀이
```python
n = input()
n = int(n)
print(n)
````
***
## 모범 답안
~~~pyhton
n = input()
n = int(n)
print(n)
~~~
