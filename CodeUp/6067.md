# 6067: [기초-출력] 출력하기67(설명)(py)
## 문제설명
```
0이 아닌 정수 1개가 입력되었을 때, 음(-)/양(+)과 짝(even)/홀(odd)을 구분해 분류해보자.
음수이면서 짝수이면, A
음수이면서 홀수이면, B
양수이면서 짝수이면, C
양수이면서 홀수이면, D
를 출력한다.
```
***
## 입력 예시
~~~
정수 1개가 입력된다.
-2147483648 ~ +2147483647, 단 0은 입력되지 않는다.
~~~
***
## 출력 예시
~~~
음수이면서 짝수이면, A
음수이면서 홀수이면, B
양수이면서 짝수이면, C
양수이면서 홀수이면, D
를 출력한다.

~~~
***
## 내 풀이
```python
if n<0 :
  if n%2==0 :
    print('A')     
  else :
    print('B')
else :
  if n%2==0 :
    print('C')
  else :
    print('D')

````
***
