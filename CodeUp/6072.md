# 6072: [기초-출력] 출력하기72(설명)(py)
## 문제설명
```
정수(1 ~ 100) 1개가 입력되었을 때 카운트다운을 출력해보자.

while 조건식 :
  ...
  ...
반복 실행구조를 사용해 보자.
```
***
## 입력 예시
~~~
정수 1개가 입력된다.
(1 ~ 100)

~~~
***
## 출력 예시
~~~
1만큼씩 줄이면서 한 줄에 1개씩 카운트다운 수를 출력한다.
~~~
***
## 내 풀이
```python
while n!=0 :
  print(n)
  n = n-1
````
***
## 모범 답안
~~~pyhton
a=int(input()) 
while a!=0: 
    print(a) 
    a=a-1
~~~
